# 🎵 SONIX — Music Player

A high-end, professional music player built with pure HTML, CSS & JavaScript.

## ✨ Features
- 🎛️ Audio Spectrum Visualizer (60-bar animated)
- 🎨 Album Art display with rotation animation
- 📋 Playlist management with search
- 🔀 Shuffle & 🔁 Repeat modes
- 🔊 Volume control with mute
- ⏱️ Progress/seek bar
- 🖱️ Drag & Drop file support
- ⌨️ Keyboard shortcuts

## ⌨️ Keyboard Shortcuts
| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `→` | +5 seconds |
| `←` | -5 seconds |
| `↑` | Volume up |
| `↓` | Volume down |
| `N` | Next track |
| `P` | Previous track |
| `S` | Toggle Shuffle |
| `R` | Toggle Repeat |
